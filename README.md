# 🍁 사용법

## 📦 설치 (최초 1회)

### Python
[python.org](https://www.python.org/downloads/) → 다운로드 → 설치 시 **"Add python.exe to PATH" 체크 필수**

### Tesseract OCR
[다운로드](https://github.com/UB-Mannheim/tesseract/wiki) → 기본 경로 그대로 설치

### 라이브러리
명령 프롬프트(cmd)에 붙여넣기:
```
pip install pyautogui pytesseract pillow opencv-python numpy keyboard pygetwindow pyperclip
```

---

## 🚀 실행

받은 파일을 한 폴더에 두고 **`실행.bat` 더블클릭**

---

## 🎮 사용 순서

### ① 게임 준비
1. 메이플 플래닛 실행 → 경매장 열기
2. 게임 창 크기를 **1282 × 752** 로 맞춤 (창 모드 최소 크기)

### ② 윈도우 정렬
프로그램 상단의 **[🎯 게임 윈도우 좌상단 정렬]** 클릭
→ 상단바가 ✅ 녹색이 되면 OK

### ③ 품목 등록 (`① 품목 설정` 탭)
| 항목 | 설명 |
|---|---|
| 검색어 | 경매장 검색창에 입력될 키워드 |
| 임계가 | 1번째 행이 이 값보다 **낮으면** 자동 구매 |
| ▲▼ | 품목 순서 변경 (위 품목부터 로테이션) |
| 🗑 삭제 | 품목 제거 |
| 전역값 | 구매 없이 N회 검색하면 다음 품목으로 |

[➕ 품목 추가]로 원하는 만큼 등록 가능.

### ④ OCR 확인
**[🔍 OCR 테스트]** 클릭 → debug 폴더 자동 열림 → 가격 인식 결과 확인

### ⑤ 동작 설정 (`② 동작 설정` 탭)
| 항목 | 기본값 | 설명 |
|---|---|---|
| 검색 반복 주기 | 1000ms | 검색 사이 간격 |
| 클릭/엔터 사이 대기 | 400ms | |
| 검색 → OCR 대기 | 350ms | |
| 구매 처리 대기 | 1500ms | 구매 확인 → 완료 모달 사이 |

### ⑥ 시작
**[▶ 시작]** 또는 **F8**

## ⏹ 정지

- **F8** / **[■ 정지]** 버튼
- 🚨 응급: 마우스를 **화면 왼쪽 위 구석**으로 휙

---

## ❓ 문제 해결

| 증상 | 해결 |
|---|---|
| 가격 인식 이상함 | 게임 창 크기 1282×752 확인 + 윈도우 정렬 재실행 |
| Tesseract 못 찾음 | 기본 경로(`C:\Program Files\Tesseract-OCR`)에 재설치 |
| Python 못 찾음 | "Add to PATH" 체크하고 재설치 |
| 모달 처리 안 됨 | 동작 설정의 **구매 처리 대기**를 2000~3000으로 증가 |
| 매크로 너무 느림 | 검색 반복 주기를 700~500으로 감소 |

## ⚠️ 주의

- 매크로 켠 동안 마우스/키보드 건드리지 말 것
- 화면 잠금/절전 시 멈춤
- 게임 창이 다른 창에 가려지면 클릭 실패
